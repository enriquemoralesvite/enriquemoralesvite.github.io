Acerca de mi

Soy un desarrollador web con formación en frontend, especializado en la creación de interfaces dinámicas y responsivas. He adquirido experiencia en tecnologías como HTML, CSS, JavaScript, Astro, Tailwind y Bootstrap, además de bases de datos con SQL Server y PostgreSQL. Me apasiona la optimización del rendimiento, la accesibilidad y la experiencia del usuario, buscando siempre soluciones eficientes y escalables. ¡Listo para transformar ideas en productos digitales!

Mis principales competencias incluyen:

  • Desarrollo Front-end: Experiencia en HTML, CSS, JavaScript, Astro, Tailwind y Bootstrap.
  • Arquitectura y Diseño CSS: Aplicación de metodologías como BEM para estilos escalables y mantenibles.
  • Manejo de Paquetes y Control de Versiones: Experiencia con npm, pnpm, Bun, Git y GitHub.
  • Gestión de Bases de Datos: Trabajo con SQL Server y PostgreSQL para administración de datos estructurados.
  • Autenticación y Seguridad: Implementación de soluciones con JWT, OAuth y Firebase.
  • Aplicaciones Web Progresivas (PWAs): Desarrollo de aplicaciones con capacidades offline y mejor rendimiento.
  • Pruebas y Depuración: Garantizando la fiabilidad del código mediante metodologías de prueba y buenas prácticas de depuración.
